# ❤️ CH Trust – Charity Website

A modern, responsive charity platform built to support social welfare initiatives by enabling donations, showcasing causes, and spreading awareness.

This platform provides a seamless digital experience for users to explore campaigns, contribute to meaningful causes, and stay connected with community-driven impact.

---

## 🌐 Live Demo

https://chtrust.sakshamjain.codes/

---

## ✨ Features

- Modern and responsive landing page  
- Donation-focused user experience  
- Campaign and cause showcase  
- Firebase Authentication for secure access  
- Firestore database for real-time data  
- Firebase Storage for media handling  
- Fast and optimized performance  
- Fully responsive across all devices  
- Serverless architecture using Firebase  

---

## 🛠️ Tech Stack

Frontend:
- Next.js  
- React  
- Tailwind CSS  

Backend / Services:
- Firebase Authentication  
- Firebase Firestore (Database)  
- Firebase Storage  

Deployment:
- Vercel (Frontend)  
- Firebase (Backend Services)

## ⚙️ Getting Started

### Clone Repository

git clone https://github.com/Sakshamjain98/ChTrust-Charity-Website.git  
cd ChTrust-Charity-Website  

---

### Setup Environment Variables

Create a `.env.local` file:

NEXT_PUBLIC_FIREBASE_API_KEY=  
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=  
NEXT_PUBLIC_FIREBASE_PROJECT_ID=  
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=  
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=  
NEXT_PUBLIC_FIREBASE_APP_ID=  

---

### Install Dependencies

npm install  

---

### Run Development Server

npm run dev  

App will run on: http://localhost:3000  

---

## 📦 Production Build

npm run build  
npm start
